# 苹果公司股价分析报告

本项目基于苹果公司(AAPL)100个交易日的股价数据，生成了一份完整的HTML分析报告。

## 数据文件
- `AAPL_100DAYS.CSV`: 包含100个交易日的股价数据，包含日期、开盘价、最高价、最低价、收盘价和成交量。

## 生成文件
1. **`苹果公司股价洞察.html`**: 完整的HTML分析报告，包含：
   - 关键指标卡片（基础指标、价格波动指标）
   - K线图 + 成交量子图（含MA5、MA20移动平均线）
   - 收盘价分布直方图与日内波动分析图
   - 数据深度分析（趋势、成交量、支撑阻力）
   - 总结与投资建议

2. **`analyze_aapl.py`**: Python分析脚本，用于计算各项指标并生成JSON数据。
3. **`aapl_metrics.json`**: 分析脚本输出的JSON格式指标数据，供HTML文件使用。

## 使用方法
1. 直接双击打开`苹果公司股价洞察.html`文件，即可查看完整报告。
2. 运行`python3 analyze_aapl.py`可重新计算指标并更新JSON数据。

## 技术栈
- **数据分析**: Python (pandas, numpy)
- **可视化**: Chart.js + chartjs-chart-financial插件
- **报告格式**: 独立HTML文件，无需外部依赖
